# ASP.NET Web Forms School Project About Chess

This is a school web development project built with ASP.NET Web Forms.
The website presents information about chess and includes user registration and an admin management page.

## Features

* Home page
* Chess rules page
* Chess openings page
* Contact page
* User registration form
* JavaScript form validation
* SQL Server LocalDB database connection
* Admin login page
* Users table for the admin
* Search users
* Delete users

## Technologies Used

* HTML
* CSS
* JavaScript
* C#
* ASP.NET Web Forms
* SQL Server LocalDB
* Visual Studio

## Project Structure

The main website files are inside the `SchoolProjectWebDev` folder.

Important parts of the project:

* `Pages` - website pages
* `Scripts` - JavaScript files
* `Content` - CSS and design files
* `images` - website images
* `App_Data` - database files
* `Site1.Master` - master pages
* `SchoolProjectWebDev.csproj`/`SchoolProjectWebDev.slnx` - project file
